# Assets

## Badge image for social sharing (LinkedIn, etc.)

Place a file named **`badge.png`** in this folder.

- **Used for:** Open Graph image when the certificate page is shared on LinkedIn, Facebook, etc. (og:image).
- **Recommended size:** 1200×630 px (LinkedIn/Facebook).
- **Content:** Your badge or certificate thumbnail so shares show a nice preview.

If `badge.png` is missing, the certificate page still works; the share preview may show no image or a generic one.
